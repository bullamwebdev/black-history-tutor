# Black History AI Tutor

An Afrofuturist AI-powered tutoring system for Black History.

Built with vanilla HTML/CSS/JS. No build step required. 250 curated questions across 50 topics and 6 historical eras.

## Features
- Adaptive Q&A with Bloom's Taxonomy levels
- Student Persona Simulator (Amara / Kofi / Zuri)
- Worked Examples with progressive exercises
- QA Pipeline Dashboard with Chart.js visualizations
- Afrofuturist × Euphoria cinematic design system
